十一月的记忆

发表于 2014-11-28   |   分类于 音十文集

晚风 ­

晚风，语于暮叶：夜，并非苦境。夜鸟有歌，秋虫吟唱，何况星星会来。 ­星星不来，有萤火虫相伴。 ­

不寂寞…… ­

­

破鞋之死 ­

破鞋之终，终漂泊而终。终于沙渚，没有墓碑。 ­

鞋的梦说：沉湎于汝之沉湎，吾心足矣…… ­

­

叶落 ­

季节的旗相遇，只是短暂的相遇。季节在风中失落了，叶也便飘零无门了。 ­

除了泥土， ­

不会有别的归宿。 ­

­

秋蝉 ­

蝉，死了。 ­

死于暮秋。枯的落叶，掩盖在蝉的尸体上，墓的形状。泥土，蝉的生命的终点。蝉声，是从宋词中出来了……

­

空空之间 ­

除了风，什么也感受不出来。­

我，生在空空之中。

不是原告也不是被告
